# Frontend Dashboard Design Specification

## Visual Style

- **Theme**: SRE fault control center, dark terminal aesthetic.
- **Background**: deep graphite `#09090b`.
- **Accents**: rose `#f43f5e` for alerts, mint `#10b981` for success, cyan `#06b6d4` for info.
- **Components**: glassmorphism panels with monospace typography.

## Layout

### Left Column — Capability Registry

- Card list of seeded capabilities.
- Highlighted tags: Chaos Tested Scenarios, Inference Cost -96%, Execution Speedup -88%.
- Code Inspector expands selected card YAML.

### Center Column — Live Sandbox & Traces

- Inject simulated Circle Wallet lockout.
- Real-time terminal trace of Consumer Agent calculus.
- Payment overlay animation for EIP-3009 authorization.
- Execution trace of YAML decision tree.

### Right Column — Savings & Ledger

- Cumulative token-cost savings.
- Real-time scrolling nanopayment ledger.
